# Flowbite Documentation

> Flowbite is a free and open-source UI component library based on Tailwind CSS and it provides ready-to-use HTML components with data attributes to enable interactive components and plugins for building modern and responsive websites.

This documentation covers all basic aspects and components of Flowbite.
